# Template Library

## Project

FieryPrompts AI

---

# Purpose

This document defines the complete Template Library used by FieryPrompts AI.

Templates provide predefined prompt structures for different domains, enabling users to generate professional prompts quickly without starting from scratch.

Each template contains recommended values, field structure, and best practices for a specific use case.
